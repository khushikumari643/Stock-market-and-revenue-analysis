# Stock-market-and-revenue-analysis
This project focuses on analyzing and visualizing stock market trends and company revenue data using Python-based data science tools. Financial datasets for Netflix and GameStop were extracted using ___yfinance___ and ___web scraping techniques___, cleaned using pandas, and visualized through interactive dashboards built with Plotly and Seaborn.
The project highlights stock price behavior, revenue growth, trading volume, moving averages, volatility, and financial trends through dynamic visualizations and analytical insights.
___Technologies Used___
1. Python
2. pandas
3. yfinance
4. BeautifulSoup
5. Plotly
6. Seaborn
7. Matplotlib

***OBJECTIVE***
To extract, clean, analyze, and visualize financial data in order to identify stock market trends, revenue patterns, and trading behavior through interactive dashboards.
